## Description

Tomato Market backend

## Version

- Spring Boot 3.1.3
- JDK 17

## Installation

## Running the app

```bash
./gradlew bootRun
nohup java -jar market-0.0.1.SNAPSHOT.jar
```

## Build

```bash
./gradlew clean build
```

## Test

```bash
./gradlew test
```

## API Test

- http://localhost:8080/swagger-ui/index.html
